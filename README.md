# cmake project demo
This is a demo for cmake project with google test and ci workflow.

# Build and Test
## Command
```
> mkdir build
> cd build
> cmake ..
> make
> make doxygen
> ./bin/unit_test
> ./bin/example_simple
```

## Script
### Build all targets including tests and examples
```
> build.sh
```
### Install library and header files of the project
```
> build.sh install
```
